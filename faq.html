Can I merge chat history from another operating system?

Yes, the .db files are the same on all platforms: Windows, OS X, Linux, Android and iOS.


How do I synchronize chat history between different computers?

You will need to get their Skype files onto one computer, merge them there, and copy the resulting file back.
Select one database on Skyperious main screen, click "Compare and merge" and choose the second database. In the comparison tab, click "Scan differences between left and right". Scanning will take some time, as it has to go through every chat in either database and compare all messages. Once ready, you can see an overview of the differences in chats and contacts, and choose to merge them into either database.
Afterwards, copy the updated file back to its Skype folder.

! The databases should belong to the same Skype account.

! Shut down Skype before starting, if one of the files is currently used by Skype.


Can I use Skyperious on OS X?

Yes, in source code form. You will need to install a number of required packages. Here is an example of doing this with Homebrew:

  • install Homebrew
  • install Python: open a terminal and run  brew install python
  • run  brew install PIL (or  brew install Pillow)
  • run  brew install pyparsing
  • install wxPython 2.9 (or 2.8), with a suitable binary from wxpython.org/download.php
The following packages are not strictly required, but provide additional functionality like Excel export or contact import:
  • brew install XlsxWriter
  • brew install Skype4Py
  • brew install python-dateutil
Download and extract the Skyperious source, and launch  skyperious.sh. Or open a terminal in the extracted source directory and run  python src\main.py.


Is there any way to keep chat history in sync automatically on all my devices?

Not really. If one of the devices is logged in all the time, it will keep the others in sync - mostly, but not always.

Merging the databases from time to time seems the simplest option.


Can I restore messages from *.dbb files that I have from an older computer?

Not directly, Skyperious does not work with *.dbb files. It is an old proprietary format that was in use prior to Skype version 4. However, Skype can automatically convert them to the current format. Try the following:

  • shut down Skype
  • move your current Skype account folder elsewhere (for example, on Windows 7, it is C:\Users\username\AppData\Roaming\Skype\skypename)
  • copy the older Skype account folder contents to the current Skype folder
  • launch Skype and log in to your account - this should convert the .dbb files to a main.db file
  • shut down Skype
  • copy the new main.db elsewhere and restore the original Skype folder


Does Skyperious open logs from other programs like AIM/ICQ/Miranda/..?

No, it only works with Skype.

If the file is an SQLite database, for example a WhatsApp message store, Skyperious will still open it. You can't browse chats or export messages, but you can access, search, and export database tables.


Do you plan to add support for doing X/Y/Z?

I might add the support, if the requested functionality seems useful, and more people contact me to ask for it.


Why are non-English letters like Å not visible in CSV export?

CSV by specification does not support non-ASCII letters, so they get converted to question marks. Export to an Excel spreadsheet will keep all characters.


Why does the database file size remain the same after deleting something?

SQLite usually wipes the deleted data clean instead of rebuilding the file on each change. If you still want the file size to go down immediately, execute VACUUM in SQL window.
